# Web Personal
Se trata de una sencilla página web, realizada cuidando al detalle los elementos html utilizados, y dándole el estilo visual utilizando únicamente css (es decir, sin utilizar frameworks como Bootstrap).

Los ficheros .svg permiten ver en forma de árbol los elementos de los documentos html. Se recomienda abrirlos utilizando Chrome, y decrementar el zoom (con Control -) para poder verlo correctamente.

**IMPORTANTE**: Los validadores de HTML5 y de CSS no funcionan cuando se abre la página en local. Es necesario que esté subida a un servidor web.
