El algoritmo de convertir un fichero json a xml fue basado en el que hay en el siguiente enlace:
https://github.com/henrikingo/xml2json/blob/master/json2xml.js

Los ficheros json se pueden abrir usando el botón de "Seleccionar archivo" o arrastrándolos al textarea de arriba.
También se puede escribir directamente en el textarea de arriba un json.

He puesto de limite al fichero json introducido 1MB, para hacer algo con la propiedad size del archivo, pero se podría quitar.
 
He incluido unos ficheros json de prueba:
-El busgijoninfo.json no deja abrirlo porque su tamaño es superior a 1MB.
-El empresas.json sí deja abrirlo.
