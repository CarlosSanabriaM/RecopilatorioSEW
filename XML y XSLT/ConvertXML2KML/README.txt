Lenguaje de programación: C#
Versión del compilador: Microsoft .NET Framework Versión 4.7.02053 (Microsoft Visual Studio Community 2017 Versión 15.4.3)
Archivos de prueba: bares.xml (entrada) y bares.kml (salida)

Instrucciones de uso: Ejecutar el ejecutable, y cuando se haya mostrado toda la información de los bares, darle a Enter para que se genere el fichero bares.kml. Ese fichero podrá ser introducido en Google Earth para observar las localizaciones de los bares.
